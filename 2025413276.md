## 三目运算符

```c++
#include<bits/stdc++.h>

using namespace std;

int a;
int x;

int main()
{
    cin >> x;
    
    a = (x > 100) ? 0 : 1;//表达式 ? 表达式 : 表达式 
    
    (x > 100) ? (a = 0) : (a = 1);
    
    return 0;
}
```
